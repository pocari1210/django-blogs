# django-blogs

<h2>◆概要◆</h2>
CRUDの操作を意識したブログサイトを構築しました。<br>
ログイン認証機能を搭載しており、<br>
ログインを行ったら、記事の投稿、投稿の削除を<br>
行えるようになっています。<br><br>
検索機能、カテゴリー機能があり、<br>
それぞれキーワードやカテゴリーなど、<br>
マッチした記事を抽出することができます。<br>

<h2>◆使用言語◆</h2>
HTML・CSS(Bootstrap)<br>
python(Django)<br>
